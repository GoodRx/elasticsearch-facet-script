h1. Script Facet Plugin for ElasticSearch

h2. Introduction

The script facet plugin provides fully scriptable facets for elasticsearch.

h2. Compatibility

The following table shows the versions of elasticsearch and jetty that Jetty Plugin was built with. 

|_. Script Facet Plugin   |_.  Elasticsearch    |
| master        |  0.19.9 -> master   |
| 1.0.0-SNAPSHOT      |  0.19.9 -> master   |
